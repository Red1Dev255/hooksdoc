Documentation des Hooks React

useState

Permet de créer un état local dans un composant fonctionnel.


const [count, setCount] = useState(0);

return <button onClick={() => setCount(count + 1)}>Clique: {count}</button>;

useReducer

Alternative à useState pour des états complexes ou multiples.


const reducer = (state, action) => {
  switch (action.type) {
    case 'increment': return { count: state.count + 1 };
    default: return state;
  }
};

const [state, dispatch] = useReducer(reducer, { count: 0 });

return <button onClick={() => dispatch({ type: 'increment' })}>{state.count}</button>;

useEffect

Effets de bord : requêtes, timers, écouteurs d'événements, etc.

Basé sur le rendu


useEffect(() => {
  console.log('Composant monté ou mis à jour');
}, [count]);

Basé sur un événement


useEffect(() => {
  const handleClick = () => alert('clicked');
  window.addEventListener('click', handleClick);

  return () => window.removeEventListener('click', handleClick);
}, []);

useLayoutEffect

S'exécute juste après le DOM est mis à jour mais avant le rendu à l'écran.


useLayoutEffect(() => {
  console.log('Avant le paint');
}, []);

useRef

Référence mutable qui ne provoque pas de re-render.


const inputRef = useRef();

const focus = () => inputRef.current.focus();

return (
  <>
    <input ref={inputRef} />
    <button onClick={focus}>Focus</button>
  </>
);

useMemo

Optimise les performances en mémorisant une valeur calculée.


const expensiveValue = useMemo(() => {
  return slowFunction(count);
}, [count]);

useCallback

Évite de recréer une fonction à chaque rendu.


const handleClick = useCallback(() => {
  alert('Clicked');
}, []);

useContext

Permet d'accéder à un contexte React.


const value = useContext(MyContext);

return <p>{value}</p>;

useTransition

Permet de marquer des mises à jour comme non urgentes.


const [isPending, startTransition] = useTransition();

startTransition(() => {
  setValue(newValue);
});

useDeferredValue

Diffère une valeur pour éviter des re-renders trop fréquents.


const deferredValue = useDeferredValue(searchInput);

useDebugValue

Affiche une étiquette personnalisée dans les outils de débogage React DevTools.


useDebugValue(user, user => user?.name || 'Anonyme');

useId

Génère un ID unique, souvent utilisé pour l'accessibilité.


const id = useId();

return (
  <>
    <label htmlFor={id}>Nom:</label>
    <input id={id} />
  </>
);
